Contract

All payloads are in JSON format.

Request

					
GET -> https://test.bit.blockpool.io/insecure/utilities/systemstatus

Response

This call does nothing but return the current service status headers for you to query. These headers are: 

					
"bit-service-method" - undefined, or a message explaining the current status if set
"bit-service-status" - "active" or "maintenance"
