Request

					
GET -> https://test.bit.blockpool.io/insecure/wallet/utilities/exchangerates

Response

As well as the supported Crypto coins we will return rates for GBP, EUR, ESD and CAD 

					
{
  "success": true,
  "rates": {
    'ticker' : { // the ticker is "adc" or "btc" or any of the supported coins
      'is_fiat': boolean,
      "rate_btc": decimal conversion rate from ticker to BTC,
      "last_update": when this rate is valid from,
      "name": the full name for the currency
    }
  }
}
