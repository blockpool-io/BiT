Working Example - BiT Supported Coins (Detail)

Click here to call the endpoint

Sample Code

This is a simple GET call.

Optionally you can return only BPL Descendant coins by adding ?descendant=bpl to the query.

Optionally you can return only BTC Descendant coins by adding ?descendant=btc to the query.

Contract

All payloads are in JSON format.

Request

					
GET -> https://test.bit.blockpool.io/insecure/wallet/utilities/supportedcoins/details

Response

					
{
  "success": true,
  "coins": [
    {
      "ticker": "the ticker",
      "isBplDescendant": boolean - is this a bpl descendant currency,
      "isBtcDescendant": boolean - is this a btc descendant currency
    }
  ]
}
